Profil Mental de Performance – A4P

Version avancée du test A4P avec 136 questions intégrées, 8 dimensions mentales, équivalence cognitive inspirée MBTI, correspondance préférences motrices, radar mental et rapport long imprimable.

Avant de commencer

Cet outil est pensé comme une base robuste pour commercialiser ton test sur GitHub Pages. Il contient déjà la structure complète pour une mise en ligne immédiate sur un dépôt vierge.

Ce que le test contient
  • 120 questions mentales
  • 16 questions cognitives
  • 8 dimensions sur 100
  • 3 profils mentaux dominants
  • Lecture inspirée MBTI
  • Préférence motrice associée
Ce que le rapport restitue
  • Indice global A4P
  • Indice pression
  • Indice stabilité
  • Radar mental
  • Axes de progression
  • Lecture jeune / parents / coach
Important : cette lecture MBTI est une équivalence de travail pour l'accompagnement. Elle ne constitue pas un MBTI officiel. La préférence motrice associée reste un repère pédagogique à confirmer sur le terrain.
